# react_vercel_app

Projeto criado para a atividade de Frontend Frameworks.

## Tecnologias
- React 18
- Vite
- react-router-dom

## Como rodar
```bash
npm install
npm run dev
```

## Requisitos atendidos
- Nome do projeto: `react_vercel_app`
- 3 telas: Home, Details, Favorites
- Consumo da API: https://potterapi-fedeperin.vercel.app/en/books/random
- Persistência de favoritos em localStorage
- Instruções de build: `npm run build`
